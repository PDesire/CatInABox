## Setup

We use the beta channel of Flutter. For highest build compatibility, you should too!

- Change to beta channel:

``flutter channel beta``

- Get all dependencies

``flutter pub get``

That's it :D

## Run

Run debug app

``flutter run``

Run release app 

``flutter run --release``
